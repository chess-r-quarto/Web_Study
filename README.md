# Web Study Portal

**[Public Access](https://chess-r-quarto.github.io/Web_Study/)**

Web Study Portal

A minimalist, dark-themed gateway for accessing specialized study modules.
Designed with the Lichess Dark Mode aesthetic, prioritizing focus and rapid access.

Visual Identity

Theme: Lichess Dark Palette (#161512 background, #bababa text).

Typography: Roboto / System UI stack for maximum legibility.

Layout: Responsive grid system mimicking a game lobby.

Interaction: Tactile hover states with specific accent colors for different module types.
